\# MVP Costeo Producción



MVP para costear órdenes de producción en una empresa pequeña de manufactura.



\## Objetivo

Calcular el costo real por lote considerando:

\- materias primas

\- mano de obra por etapa

\- tiempo de máquina

\- costos indirectos



\## Stack

\- Python

\- FastAPI

\- SQLite

\- HTML templates
